# Launch Post Draft

PanePilot is a tiny tmux layer for terminal-native coding agents.

It does one thing well: keep an agent session attached to a real workspace so
you can detach, reattach, inject tasks, and automate prompts without rebuilding
context every time.

What it gives you:

- persistent tmux-backed agent sessions
- one-command task injection
- keepalive and recovery scripts
- cron-friendly automation
- runtime-agnostic config for Codex, Claude, OpenClaw, and similar CLIs

It is not an orchestrator or hosted platform. It is the missing operational
layer between a terminal agent and a real machine.
