# Hacker News Draft

Title:
Show HN: PanePilot, a tmux workspace layer for terminal coding agents

Body:

PanePilot is a small shell tool for running terminal-native coding agents as
durable tmux workspaces.

It adds a control surface around the repetitive parts of daily usage:

- start or reattach an agent workspace
- inject tasks without entering the pane
- inspect health and readiness
- recover unhealthy sessions
- keep runtime-specific config out of the repo

I verified the current profiles with Codex and OpenClaw on a real machine.

https://github.com/Galaxy-0/panepilot
