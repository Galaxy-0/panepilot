# X Post

PanePilot is a tiny tmux layer for terminal-native coding agents.

It keeps one agent pinned to one real workspace so you can:

- detach and reattach later
- inject tasks from outside the pane
- check health and readiness
- recover unhealthy sessions without blind restarts

Verified profiles currently included for Codex and OpenClaw.
